Copy these environment variables to a new file named ".env" in your project root:

# Gemini AI API Configuration
GOOGLE_GENERATIVE_AI_API_KEY=your_gemini_api_key_here

# Database Configuration
DATABASE_URL=postgresql://postgres:password@localhost:5433/auth_chat_db

# Auth.js secret key (generate a random string)
AUTH_SECRET=your_auth_secret_here

# Gmail SMTP Settings
SMTP_HOST=smtp.gmail.com
SMTP_PORT=587
SMTP_USER=your_email@gmail.com
SMTP_PASS=your_app_password_here
SMTP_FROM=your_email@gmail.com

# Google OAuth (get from Google Cloud Console)
AUTH_GOOGLE_ID=your_google_oauth_client_id_here
AUTH_GOOGLE_SECRET=your_google_oauth_client_secret_here

# GitHub OAuth (get from GitHub Developer Settings)
AUTH_GITHUB_ID=your_github_oauth_client_id_here
AUTH_GITHUB_SECRET=your_github_oauth_client_secret_here

# Application URL
AUTH_URL=http://localhost:5173

===========================================
SETUP INSTRUCTIONS:
===========================================

1. Create a new file named ".env" in your project root
2. Copy all the variables above into that file
3. Replace placeholder values with your actual credentials
4. Save the file
5. Run: docker-compose up -d
6. Run: pnpm install
7. Run: pnpm db:push
8. Run: pnpm dev

Your app will be available at: http://localhost:5173
Database will be available at: localhost:5433
Redis will be available at: localhost:6379

===========================================
HOW TO GET CREDENTIALS:
===========================================

1. Google OAuth:
   - Go to https://console.cloud.google.com/
   - Create a new project or select existing
   - Enable Google+ API
   - Create OAuth 2.0 credentials
   - Add http://localhost:5173/auth/callback/google to authorized redirect URIs

2. GitHub OAuth:
   - Go to https://github.com/settings/developers
   - Create a new OAuth App
   - Add http://localhost:5173/auth/callback/github to callback URL

3. Gemini AI API:
   - Go to https://makersuite.google.com/app/apikey
   - Create a new API key

4. Gmail App Password:
   - Enable 2-factor authentication on your Gmail account
   - Generate an app password for this application
